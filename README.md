### Developer Notes
1. Download and install [Java JDK 1.7.0_55](http://www.oracle.com/technetwork/java/javase/downloads/jdk7-downloads-1880260.html).
1. Download and install [Apache Maven](https://maven.apache.org/).
1. Download and install [git](https://git-scm.com/).
1. Clone this repository.
    
    ````
    git clone https://github.com/AigenVector/dawdle.git
    ````
    
1. Give it a build.
     
    ````
    mvn package
    ````
     
1. Grant the start script privilege.
    
    ````
    chmod 0755 start.sh
    ````
    
1. Give it a run.
    
    ````
    ./start.sh
    ````
    
1. Happy hacking!!!
